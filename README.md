# NewsScraper
Scraper for news sites made in spanish.

When pipeline.py is run, it saves a .db file for each website in the Databases folder.

## Currently supports:

* El universal
* El pais
* Cnnespanol
* Pagina12
* Milenio

## Libraries needed:

* Pandas
* hashlib
* nltk
* sqlalchemy
* requests
* lxml
* pyyaml
